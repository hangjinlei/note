# CLI tools

- new

  ```shell
  ng new angular-tour-of-heroes
  ```

- run

  ```shell
  ng serve --open
  ```

- component

  ```shell
  ng generate component <component-name>
  ng g c <component-name>
  ```

- service

  ```shell
  ng generate service <service-path>
  ng g s <service-path>
  ```
