# Ruby on Rails

## 安装 Rails

- Ruby
  - `ruby --version`
- SQLite3
  - `sqlite3 --version`
- Node.js
  - `node --version`
- Yarn
  - `npm install --global yarn`
  - `yarn --version`
- Rails
  - `gem install rails`
  - `rails --version`

## 创建项目

- `rails new blog`
- `rails new blog --api`
- `rails new blog --skip-test-unit`

## 启动

`rails s` / `rails server`
