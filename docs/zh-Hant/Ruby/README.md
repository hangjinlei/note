## 命名规范

## Ruby 命名规范

| 名称       | 示例        |
| :--------- | :---------- |
| class_name | `ClassName` |
| 局部变量   | `stu_name`  |
| 常量       | `Stu_Count` |

## Rails 命名规范

| 名称       | 示例                | 注意 |
| :--------- | :------------------ | :--- |
| controller | `models_controller` | 复数 |
| model      | `model`             | 单数 |
